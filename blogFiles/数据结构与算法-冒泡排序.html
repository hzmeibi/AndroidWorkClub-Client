概念

冒泡排序，是一种排序算法。它重复地走访要排序的队列，一次比较两个相邻的元素，如果顺序错误就把它们交换过来，直到没有需要交换的元素。冒泡排序主要体现出来的就是换位思想。

原理

  1. 比较相邻的元素。如果第一个比第二个大，就交换他们两个。
  2. 对每一对相邻元素做同样的工作，从开始第一对到结尾最后一对。
  3. 针对所有元素重复以上步骤，直到没有任何一对元素需要交换为止

代码实现

/**
 * Created by milo on 2018/6/3.
 * 排序
 */

public class Sort {

    /**
     * 冒泡排序 升序排列
     */
    public void BubbleSort() {
        int[] ids = {1, 3, 2, 0, 5, 4};
        System.out.println("排序前：");
        for (int i = 0; i < ids.length; i++) {
            System.out.println(ids[i] + ",");
        }
        int editCount = ids.length;//用于跳出循环
        while (editCount > 0) {
            editCount = 0;
            for (int i = 0; i < ids.length - 1; i++) {
                int id1 = ids[i];
                int id2 = ids[i + 1];
                //比较第i个元素和第i+个元素的大小,不满足条件交换为止，
                if (id1 > id2) {
                    //记录交换的的次数，当次数为0时说明已经排序完成及跳出循环
                    editCount++;
                    ids[i] = id2;
                    ids[i + 1] = id1;
                }
            }
        }

        System.out.println("排序后：");
        for (int i = 0; i < ids.length; i++) {
            System.out.println(ids[i] + ",");
        }

    }

    public static void main(String[] arg) {
        Sort sort = new Sort();
        sort.BubbleSort();
    }

}

运行结果：

排序前：
1,3,2,0,5,4,
排序后：
0,1,2,3,4,5,
